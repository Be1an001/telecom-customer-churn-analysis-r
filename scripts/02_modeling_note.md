## How to Read the Modeling Section

The modeling content should be treated as part of the course-project record, not as a production-ready churn model. The local repository does not currently include a clean standalone modeling script with a full reproducible train/test workflow.

The final report discusses logistic regression, lasso, ridge, and ROC/AUC results, including an AUC value reported in the group submission. Before using the modeling work for stronger data science claims, it would need clearer validation steps, leakage checks, baseline comparison, and a reproducible script.

This note is included for transparency because this was a group academic project and because my strongest direct contribution in this portfolio version is the R-based exploratory analysis and business interpretation.
