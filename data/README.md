# Data Note

This public portfolio version does not include the raw CSV files.

The original course project used two files:
- `telecom_customer_churn.csv`
- `telecom_zipcode_population.csv`

The analysis merged the two datasets by `Zip.Code`.

If you want to reproduce the analysis locally, create a folder named `raw` inside `data/` on your own computer and place the original files there:

- `data/raw/telecom_customer_churn.csv`
- `data/raw/telecom_zipcode_population.csv`

The final report describes the original project data as:
- a telecom customer churn table with 7,043 customer records
- a zipcode population table used to add geographic context
- a merged dataset with 39 columns
